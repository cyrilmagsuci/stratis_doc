Getting Started
###############

The Stratis Full Node is completely Open Source and can be fully
inspected on our GitHub project page. If you would prefer to build from
source, please follow the below:

https://github.com/stratisproject/StratisFullNode/blob/master/Documentation/getting-started.md

This document will focus on running the Stratis Full Node and creating a
significant amount of addresses, returning those addresses and
transferring funds between them.

For this document, built binaries will be referenced, follow the below
steps to obtain the published binaries for the current release.
